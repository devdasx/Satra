Before you write

Satra is self-custody software. We cannot recover lost recovery phrases, reverse blockchain transactions, reset private keys, or restore funds from an exposed backup. If you are reporting a transaction issue, include only public transaction hashes, public wallet addresses, asset symbol, network, and app version.

For source code and project documentation, visit github.com/devdasx/Satra.

Satra is open-source, self-custody wallet software. It does not provide exchange, brokerage, custody, banking, staking, lending, money transmission, or investment advisory services.
