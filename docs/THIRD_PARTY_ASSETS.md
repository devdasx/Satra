# Third-Party Assets

## Fonts

Satra uses the following Google Fonts files in the Android app:

| Family | File | Source | License |
|--------|------|--------|---------|
| Outfit | `app/src/main/res/font/outfit_variable.ttf` | Google Fonts, `ofl/outfit` | SIL Open Font License 1.1 |
| Space Grotesk | `app/src/main/res/font/space_grotesk_variable.ttf` | Google Fonts, `ofl/spacegrotesk` | SIL Open Font License 1.1 |

License text is stored in `docs/licenses/OFL-Outfit.txt` and `docs/licenses/OFL-Space-Grotesk.txt`.

The Satra brand kit in `brand/satra-brand-kit` is project-owned brand material.
