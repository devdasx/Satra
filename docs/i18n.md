# Localization Plan

English is the source language while Satra is still being designed. Every new user-facing string must be added to `app/src/main/res/values/strings.xml` immediately, even before translated resource folders are created.

When the whole app is ready for localization, translate the complete string catalog for these target languages based on the 2026 total-speaker ranking:

1. English (`en`)
2. Mandarin Chinese (`zh-Hans`)
3. Hindi (`hi`)
4. Spanish (`es`)
5. Standard Arabic (`ar`)
6. French (`fr`)
7. Bengali (`bn`)
8. Portuguese (`pt`)
9. Indonesian (`id`)
10. Urdu (`ur`)
11. Russian (`ru`)
12. German (`de`)
13. Japanese (`ja`)
14. Nigerian Pidgin (`pcm`)
15. Egyptian Arabic (`arz`)
16. Marathi (`mr`)
17. Vietnamese (`vi`)
18. Telugu (`te`)
19. Swahili (`sw`)
20. Hausa (`ha`)
21. Turkish (`tr`)
22. Western Punjabi (`pnb`)
23. Tagalog/Filipino (`fil`)
24. Tamil (`ta`)
25. Yue Chinese/Cantonese (`yue-Hant`)

Some language rankings treat macrolanguages and regional varieties differently. Keep this list stable for product planning unless the user asks to revise it.
